## 💀 这是什么鬼？

**AI 时代代码太多了，写完就丢，丢哪儿？**

丢进 **代码尸体集中营** 🪦

- 📦 你不管的代码 → AI 自动扫描 → 丢进墓地
- 🔍 新项目需要功能 → AI 去墓地翻一翻 → 诈尸复用
- 🧟 死掉的代码不是真死，只是等 AI 来捞

> **核心哲学：** 墓地不是终点，是代码的"轮回中转站"

## 🚀 快速开始

### 1. 安装

```bash
npm install -g code-corpses
```

### 2. 配置 Agent（一次就好）

```bash
code-corpses init
# 授权 GitHub → 设置扫描频率 → 搞定
```

### 3. 让 Agent 当墓地管理员

Agent 会自动：
- ✅ 定期扫描 → 找到死代码 → 生成墓碑
- ✅ 检测诈尸 → 旧代码被复用 → 通知你
- ✅ 统计报告 → 发送到你的渠道

### 4. 忘记它 🧠

**Human off the loop：** 授权一次，AI 自己搞定一切
